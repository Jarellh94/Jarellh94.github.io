Currently Working On...

Unnamed 2D Platformer - New Enemies

August 8, 2018 - I added 3 new enemies. A large strong enemy that takes multiple hits and does more damage. He uses the same movement scripts as a normal roaming enemy. A variation on the shooting enemy that targets the player. It patrols and uses a raycast to see the player. Once it does, it stops and starts shooting. A grenader type enemy that sits in the ground and throws projectiles. In adding this enemy, I added a new throwable projectile type. Each projectile has a variable on whether or not it is throw. If thrown, then when fired force is added vertically as well as horizontally, producing an ark handled by the physics engine.

Unnamed 2D Platformer - Level Hazards and Falling Platforms

August 8, 2018 - I recently started working on more interesting level mechanics. I've created a class for level hazards that deal damage to the player. They can be of a few different types, such as water, falling rocks, and spikes. I've also added falling platforms. They work by placing a platform spawner in the level that spawns a new platform after a set time passes after the previous platform falls. The platform respawn speed and falling speed can aslo be customized per platform. To the left is a small bug that caused platform spawners to spawn platforms constantly. Producing a Platform Fountain.

Unnamed 2D Platformer - Added Armor Pickup and Shooting Enemy

June 12, 2018 - 2D Platformer now has equippable armor. Armor adds another health point to the player. Currently no UI representation. I also added an enemy that fires prefabbed projectiles. New enemy uses new EnemyShooter class that functions similar to the player's gun class and allows for multiple kinds of projectiles to be fired. Also added physics to the bullets to allow for dropping.

Unnamed 2D Platformer - Adding Webgun and Honey Collectable

June 10, 2018 - 2D Platformer now has a new weapon and another collectable. The web gun has been added. It stops enemies for a short time. Still needs to be tweaked for usefulness and balance, and given proper art and effects. Can be good for stopping an enemy for a higher jump to an unreachable area. Currently the weapons work on a base equippable class that stores all the information about the weapon, making adding new weapons easier.
